## What I Care About

Quality is not a test phase.  
It is a delivery system design problem.

I believe:
- Automation should reduce cognitive load, not add to it.
- CI/CD pipelines are policy engines, not just build runners.
- Quality metrics must inform business decisions, not decorate dashboards.
- Directors of Quality should think in systems, not test cases.
